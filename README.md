# Cool-Movie-Recommender馃幀
Simple Movie Recommendation System using Python and Pandas 

## Features
- Filter movies by genre
- Choose how many recommendations to display
- Works with a dataset of 100+ movies

## How to Run
1. Install pandas:
pip install pandas


2. Run the program:
python movie_recommender.py


## Example
Enter movie genre: sci-fi  
How many movies do you want? 5  

Top 5 movies in sci-fi:
- interstellar
- the matrix
- avatar
- gravity
- arrival
